Clima

Essa aplicação web foi criada para proporcionar aos seus usuários, dados metereológicos em tempo real, providos pelo serviço da OpenWeather.


Sobretudo, este é um projeto desenvolvido em HTML, CSS, Javascript e Python por meio do framework Flask. Por mais simples que pareça, foi muito importante para consolidar conhecimentos e novas habilidades ao criador.
